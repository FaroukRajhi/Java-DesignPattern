Allows two incompatible interfaces to work together.
Used when the client expects a (target) interface.
The adapter class allows the use of the available interface and the target interface.

=> allows objects with incompatible interfaces to collaborate. 
It acts as a bridge between two incompatible interfaces.

**Key components**

Target: The interface that the client expects

Adaptee: The existing class that needs adapting

Adapter: The class that bridges the Target and Adaptee

Client: The class that uses the Target interface

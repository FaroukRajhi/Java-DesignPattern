Used to manipulate classes and everything in a class.
Can slow down a program beacause the JVM can't optimize the code
Can't be used with applets

=> allows runtime inspection and modification of classes, interfaces, fields, methods, and constructors without knowing their names at compile time.


Enables to:
Examine class structure at runtime

Invoke methods dynamically

Access private fields and methods

Create instances of unknown classes

Modify accessibility of class members
